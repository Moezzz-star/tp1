HTML Course

This course will introduce you to the basics of HTML (Hypertext Markup Language), the standard language for creating web pages.

Course Description

What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating web pages.
  • HTML describes the structure of a web page.
  • HTML consists of a series of elements.
  • HTML elements tell the browser how to display the content.
  • HTML elements label pieces of content such as headings, paragraphs, and links.

Learn how to structure web pages using HTML elements, create links, add images, and more!
